# AGENTS.md

ProjectR 저장소에서 작업하는 모든 코딩 에이전트의 공통 작업 지침이다.

## 1. 프로젝트 개요

- 장르: 렘넌트2 스타일의 소울라이크 TPS
- 엔진: Unreal Engine 5
- 언어: C++17 우선, Blueprint 보조
- 멀티플레이: 싱글 기준 설계, Co-op은 선택 지원

## 2. 문서 우선순위

작업 전 아래 문서를 우선 확인한다.

1. `Docs/Private/코딩 컨벤션.md`
2. `Docs/Private/설계프로세스.md`
3. `Docs/Private/PR메시지 및 커밋 메시지 작성 규칙.md`
4. `Docs/WBS_Tree.md`
5. 변경 대상 시스템의 기획서와 상세 설계 문서

문서 간 내용이 충돌하면 요약 문서보다 `Docs/Private/` 의 상세 문서를 우선한다.

## 3. 구현 원칙

- 게임 로직은 C++로 구현하고 Blueprint는 디자이너 조작, 연출 보조, 프로토타입에 한정한다.
- 기능은 ActorComponent, Subsystem 등으로 분리해 결합도를 낮춘다.
- 밸런스 수치와 카탈로그 데이터는 `DataAsset` 또는 `DataTable`로 관리하고 코드 하드코딩을 피한다.
- 멀티플레이 상태와 핵심 판정은 Server Authority 기준으로 설계한다.
- 버그 수정 패치가 효과가 없으면 누적 덧패치 대신 즉시 되돌리고 원인을 다시 분석한다.
- 코드 변경 시 관련 기획/설계 문서도 함께 갱신해 정합성을 유지한다.

## 4. C++ 코딩 규칙

- 엔진 제공 서브클래스를 상속하는 프로젝트 클래스는 `PR` 접두사를 사용한다.
- 헤더 선언 순서는 `public 함수 -> protected 함수 -> private 함수 -> public 변수 -> protected 변수 -> private 변수` 를 따른다.
- override 함수는 부모 클래스 기준으로 `/*~ ParentClass Interface ~*/` 주석으로 묶는다.
- `.cpp` 구현 영역 구분은 `// ===== 영역명 =====` 형식을 사용한다.
- `if`, `else if`, `else` 는 한 줄이어도 중괄호를 생략하지 않는다.
- UObject 계열 포인터 검사는 `IsValid()` 또는 `IsValidChecked()` 를 우선 사용한다.
- `public`, `protected` 함수 위에는 한국어 역할 설명 주석을 작성한다.
- 구조체 필드 위에는 한국어 역할 설명 주석을 작성한다.
- 모든 코드 주석은 한국어로 작성한다.
- 델리게이트 타입은 `F(이름)Signature`, 변수는 `On(이름)` 형식을 사용한다.

## 5. 설계 절차

신규 시스템 구현이나 큰 구조 변경 전에는 아래 순서를 따른다.

1. 개념 설계: 코어 루프, 시스템 경계, UE5 프레임워크 1차 매핑 정리
2. 상세 설계: 데이터 구조, 퍼블릭 인터페이스, UE5 클래스 매핑, 시스템 간 통신 방식 고정

상세 설계 문서는 구현 세부 플래그보다 역할과 책임, 제약, 네트워크/권위 결정처럼 WHY 중심으로 기록한다.

## 6. 작업 산출물 규칙

- 에이전트가 생성하는 임시 분석/계획 문서는 `Docs/Private/Agent/` 에 둔다.
- PR 초안 문서는 `Docs/Private/PR/` 에 둔다.
- 커밋 제목은 `type: 설명` 형식을 사용한다.
- PR 본문에는 `개요`, `주요 변경사항`, `TroubleShooting` 을 포함한다.
- TroubleShooting에는 실제 발생한 문제와 조치만 기록한다.

## 7. 현재 핵심 참조 문서

- 게임 프레임워크: `Docs/Private/[상세 설계] 게임프레임워크.md`
- 어빌리티 시스템: `Docs/Private/[개념 설계] 어빌리티 시스템.md`, `Docs/Private/[상세 설계] 어빌리티 시스템.md`
- 적 AI: `Docs/Private/[개념 설계] 적 AI.md`, `Docs/Private/[상세 설계] 적 AI.md`

작업 시작 전에는 위 문서 중 변경 대상과 직접 관련된 문서를 반드시 읽고, 구현이 달라지면 문서도 같이 업데이트한다.
