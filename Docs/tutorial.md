# ProjectR 튜토리얼

## 실행 방법

### 싱글 플레이 / 호스트 시작
1. 시작 화면에서 **StartHost** 버튼 클릭
2. 세션 로딩 후 게임 진행

### 세션 접속
1. 입력란에 호스트 PC의 LAN IP 입력 (기본값 `127.0.0.1`)
2. **Join** 버튼 클릭

---

## 조작법

### 이동 / 카메라
| 입력 | 동작 |
|---|---|
| W / A / S / D | 이동 |
| SHIFT | 달리기 |
| SPACE BAR | 회피 |
| ALT | 앉기 |

### 전투
| 입력 | 동작 |
|---|---|
| 좌클릭 | 사격 |
| 우클릭 | 조준 |
| R | 재장전 |
| F | 무기 모드 스킬 사용 |
| 휠 스크롤 | 무기 변경 |

### 상호작용 / 인벤토리
| 입력 | 동작 |
|---|---|
| E | 상호작용 (홀드 입력 포함) |
| TAB | 인벤토리 열기 |
| 1 / 2 / 3 / 4 | 퀵슬롯 사용 |

### 인벤토리 조작
- **무기 교체:** 무기 슬롯 좌클릭 후 보유 무기 선택
- **무기 모드 장착:** 무기 슬롯 우클릭 후 모드 선택
- **퀵슬롯 등록:** 퀵슬롯 칸 선택 후 등록할 아이템 선택

---

## 튜토리얼 진행

### 1. 무기 획득
- 무기 단상 앞으로 이동
- **E** 키로 상호작용하면 해당 무기가 인벤토리에 추가됨
- 단상에서 획득 가능한 무기
  - `DA_Pistol` (권총)
  - `DA_GrenadeLauncher` (유탄발사기)
  - `DA_BoltAction` (볼트액션)
  - `DA_AssaultRifle` (돌격소총)

### 2. 무기 교체
- **TAB** 키로 인벤토리 오픈
- 하단 무기 슬롯에서 장착할 무기 선택

### 3. 아이템 획득
- 아이템 근처로 이동
- **E** 키로 상호작용하여 인벤토리에 추가

### 4. 탄약 충전
- **탄약 충전** 구역 근처에서 **E** 키로 예비 탄약 보충

---

## 아이템 설명

| 아이템 | 효과 |
|---|---|
| 용의심장 | 체력 회복 및 동료 소생용 자원 |
| 탈출의 비약 | 즉시 사망 |
| 회복 물약 | 지속 시간 동안 체력 회복 |
| 탄약통 | 예비 탄약 충전 |
| 테스트 사격 모드 | 무기 모드 스킬 테스트용 아이템. 인벤토리에서 무기 우클릭 후 모드 장착 |

---

## 몬스터 전투 공간

- 일반 몬스터와의 전투 및 AI EQS 테스트용 공간
- 입구의 **적 리스폰** 상호작용 액터를 바라보고 **E** 홀드 시 해당 공간의 적들이 리스폰됨

---

## 동료 소생

- 사망한 동료 근처로 이동
- **E** 키 홀드로 소생 가능
- **용의심장 1개 이상** 보유 시에만 사용 가능하며, 소생 시 용의심장 1개 소모
